# Maryland Cannabis Dispensaries
Project collab with Lawrence Brown, PhD, MPA  - [Black Butterfly Dream Lab](https://theblackbutterflyproject.com/))

Interactive map tool to visualize cannabis dispensaries across state of Maryland, 2022-2025.
