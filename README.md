# Maryland Cannabis Dispensaries
Project collab with Dr. Lawrence Brown of Morgan State University and the [Black Butterfly Dream Lab](https://theblackbutterflyproject.com/).

Interactive map tool to visualize cannabis dispensaries across state of Maryland, 2022-2025. 

[Check out the map with 2022 data](https://studio.foursquare.com/map/public/ce38ff1a-252b-494f-a599-4f42c797b2f0). [updated data to come - WIP as of 12/2025]
